Wasteless Bites

Every day restaurants have an excess amount of food at the end of each day and throw away all the leftovers. Wasteless Bites can help prevent unnecessary waste and help those that might be in need for free. Restaurants can use the website to post food for people instead of throwing it away at the end of day. It is also a great way to raise awareness about their surplus food items and attract potential customers.

Collaboration Project

Website: Wasteless Bites , Github Repo

Description: Wasteless Bites is an impactful platform dedicated to reducing food waste and creating a sustainable food ecosystem. Our website offers a user-friendly interface and aims to inspire positive change and contribute to a more sustainable future.

Project Requirements

  • Utilize PERN stack, Redux, Cloudinary and React Leaflet to develop the web application.
  • Develop and deploy a secure registration and authentication system for users and businesses.
  • Implement a user-friendly interface with intuitive navigation and interactive features to enhance the user experience.
  • Utilize geolocation services to provide users with accurate and relevant information about nearby food options.

    • Project Overview

    The Wasteless Bites project is a web application developed using the PERN stack, Redux, Cloudinary, and React Leaflet. With a focus on reducing food waste, the application offers a platform for users and businesses to connect and share surplus food items. The project incorporates a secure registration and authentication system to ensure the safety of user information. Additionally, it features a user-friendly interface with intuitive navigation and interactive elements, enhancing the overall user experience. By utilizing geolocation services, Wasteless Bites provides users with accurate and relevant information about nearby food options, promoting convenience and accessibility.
